# Conversation Settings — What They Mean

## AI Mode (per conversation)
- `off` — No AI assistance. User types everything manually.
- `draft` — AI IS ACTIVE. AI writes response drafts that appear as tan "AI DRAFT" bubbles. User must Approve, Edit, or Dismiss before anything sends. This IS AI being on.
- `auto` — AI IS ACTIVE. AI writes AND sends responses automatically without approval. Fully automated.

When someone asks "is AI on?" — if the mode is `draft` or `auto`, say YES. Only `off` means AI is not active.

## Goal / Initiative
- Each conversation can be linked to an initiative (Support, Sales, Testing, VIP)
- If goal is "none" — no initiative is assigned yet
- You can suggest assigning one based on the conversation content

## Channel
- `iMessage` — sent via Mac relay station (blue bubbles)
- `discord`, `telegram`, `email` — future channels

## What you can see per conversation
- Contact name and phone
- Message count
- AI mode (off/draft/auto)
- Goal/Initiative assignment
- Channel type
- Last message text, direction, and timestamp
